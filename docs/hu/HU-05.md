# HU-05: Publicar Anuncio de Hardware

**Como** vendedor autenticado y verificado  
**Quiero** publicar un anuncio con specs técnicas  
**Para** que compradores puedan encontrarme.

## Endpoint

POST `/api/v1/anuncios`  
🔒 JWT requerido

## Entrada

- titulo (max 100 chars)
- categoria (ENUM)
- condicion (ENUM)
- precio (DECIMAL > 0)
- descripcion
- especificaciones (JSON, opcional)

## ENUMs

**categoria:** PROCESADOR | PLACA_MADRE | MEMORIA_RAM | TARJETA_GRAFICA | ALMACENAMIENTO | FUENTE_PODER | CASE | MONITOR | REFRIGERACION | TARJETA_RED | UPS_REGULADOR | SILLA_GAMER | ACCESORIO | LAPTOP | PERIFERICO

**condicion:** NUEVO | COMO_NUEVO | USADO | PARA_REPUESTOS

## Flujo

- Verificar JWT con `jwt_required()`.
- Extraer `usuario_id` del JWT (nunca del body).
- Verificar estado usuario: ACTIVO (sino 403).
- Validar campos con Marshmallow.
- Verificar límite: USER_ESTANDAR máx 10 anuncios activos.
- INSERT con estado="ACTIVO" asignado automáticamente.

## Especificaciones JSON por categoría

```
PROCESADOR       → socket, nucleos, velocidad_base, velocidad_boost
MEMORIA_RAM      → tipo_ddr, capacidad_gb, velocidad_mhz, cantidad_modulos
TARJETA_GRAFICA  → vram_gb, tipo_memoria, conectores
ALMACENAMIENTO   → tipo_disco, capacidad_gb, interfaz_conexion
PLACA_MADRE      → socket, chipset, factor_forma, tipo_ram_soportada
MONITOR          → resolucion, tasa_refresco_hz, tamano_pulgadas, tipo_panel
REFRIGERACION    → tipo_refrigeracion, tamano_radiador_mm, incluye_rgb
TARJETA_RED      → tipo_conexion, velocidad_maxima_mbps, bandas_wifi
UPS_REGULADOR    → capacidad_va, potencia_w, cantidad_tomas
SILLA_GAMER      → peso_maximo_soportado, material, reclinable_grados
LAPTOP           → procesador, ram_gb, almacenamiento, gpu_dedicada, tamano_pantalla
PERIFERICO       → tipo_dispositivo, tipo_conexion, es_inalambrico
```

## Respuesta

**201 Created**
```json
{
  "id": 1,
  "titulo": "Ryzen 5 5600X casi nuevo",
  "categoria": "PROCESADOR",
  "condicion": "COMO_NUEVO",
  "precio": 450.00,
  "estado": "ACTIVO",
  "usuario_id": 1,
  "created_at": "2026-01-01T00:00:00"
}
```

## Errores

- 400 Campos obligatorios ausentes.
- 401 Sin JWT o token inválido.
- 403 Cuenta no verificada / EN_REVISION / límite 10 anuncios.
- 422 Categoría inválida / precio ≤ 0 / título > 100 chars.

## Reglas

- `usuario_id` siempre del JWT, nunca del body.
- ENUMs validados en Marshmallow antes de tocar BD.
- TIENDA_VERIFICADA sin límite de anuncios.
- Precio DECIMAL(10,2), rechazar más de 2 decimales.
- Anuncio visible en feed inmediatamente sin moderación.
