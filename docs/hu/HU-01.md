# HU-01: Registro de Usuario Estándar

**Como** ciudadano de Ayacucho  
**Quiero** registrarme con correo y contraseña  
**Para** publicar y explorar anuncios de hardware.

## Endpoint

POST `/api/v1/auth/register`

## Entrada

- nombre
- correo
- password
- telefono

## Flujo

- Validar con Marshmallow.
- Verificar correo único en BD.
- Cifrar con bcrypt (salt=10).
- Crear usuario vía SQLAlchemy.
- Rol: USER_ESTANDAR.
- Estado: PENDIENTE_VERIFICACION.

## Respuesta

**201 Created**

```json
{
  "id": 1,
  "nombre": "Juan Quispe",
  "correo": "juan@gmail.com",
  "rol": "USER_ESTANDAR"
}
```

## Errores

- 400 Campos inválidos.
- 409 Correo duplicado.
- 422 Contraseña inválida.

## Reglas

- Correo único (constraint BD).
- Teléfono 9 dígitos.
- Nunca exponer password.
