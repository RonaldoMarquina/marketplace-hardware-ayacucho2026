# Contexto Codex

Esta carpeta agrupa material auxiliar para trabajo asistido por IA dentro del
repositorio.

## Uso

- `legacy-prompts/`: prompts y contexto historico usados durante fases previas
  del desarrollo
- este espacio puede crecer con material auxiliar para Codex que no pertenezca
  a la documentacion principal

## Regla

Este material no reemplaza la documentacion oficial de `docs/`.
