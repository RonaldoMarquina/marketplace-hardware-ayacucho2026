# OpenSpec

Esta carpeta contiene los artefactos operativos reales de OpenSpec.

## Contenido esperado

- `config.yaml`: configuracion base de OpenSpec
- `changes/`: cambios propuestos, en curso o archivados
- `specs/`: especificaciones gestionadas por OpenSpec

## Especificaciones activas actuales

- `specs/auth-security.md`
- `specs/marketplace-core.md`
- `specs/admin-moderation.md`

## Regla

Si un artefacto pertenece al flujo de la herramienta OpenSpec, debe vivir aqui.
Si solo explica el enfoque adoptado en el proyecto, debe vivir en
`docs/openspec/`.
